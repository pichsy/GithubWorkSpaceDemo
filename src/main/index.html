关于 xwidget

`xwidget` 是一个强大且灵活的小部件库，旨在简化交互式Web应用程序的开发。它提供了广泛的可定制小部件，可以轻松集成到任何项目中。

主要特点：

  • 易于使用和集成
  • 高度可定制
  • 轻量且快速
  • 详尽的文档和支持

**使用方法**：

**安装**： 
npm install xwidget

**基本用法**： 
import { Widget } from 'xwidget';
const myWidget = new Widget({ /* options */ });
myWidget.render(document.getElementById('widget-container'));

**高级用法**： 
const myWidget = new Widget({
    color: 'blue',
    size: 'large',
    onClick: () => alert('Widget clicked!')
});
myWidget.render(document.getElementById('widget-container'));
